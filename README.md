# ODF Base - Market Regimes Guide

Comprehensive study guide on regime detection, Hidden Markov Models, volatility analysis, and trading applications.

## Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS 4
- **UI:** React 19
- **Graph:** Cytoscape.js + cola layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Features

- **Home** - Hero + Curriculum overview
- **Brain** - Interactive knowledge graph
- **Glossary** - Searchable term index
- **Dashboard** - Module access (Discord-gated)

## Deployment

Deploy on Vercel - zero config.
